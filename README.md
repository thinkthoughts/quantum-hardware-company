# quantum-hardware-company

Triplet-Phase-Lock (CGCS) based hardware and control repository for ion/quantum trap system design, fabrication workflows, calibration routines, noise mitigation, and control-stack integration.

## Repository Structure

- `electrode-geometries/` — Electrode CAD, parametric geometry definitions, simulations, and layout variants.
- `fabrication/` — Process notes, masks, tolerances, vendor files, and fabrication run documentation.
- `trap-layouts/` — Trap architecture layouts, zone maps, routing, and physical design references.
- `control-electronics/` — DAC/ADC systems, RF/DC electronics, timing hardware, schematics, and board docs.
- `calibration/` — Calibration procedures, datasets, fitting scripts, and validation reports.
- `noise-mitigation/` — Noise models, filtering strategies, shielding notes, and characterization results.
- `control-stack/` — Software control interfaces, pulse sequencing, device APIs, and experiment orchestration.
